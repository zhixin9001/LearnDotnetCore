制作自己的镜像
  之前的做法只是当作虚拟机
  用docker做服务器，即抛，不需要有vi等编辑器
  不应该在容器中写代码、敲指令
  Dockerfile
  不应该在每个容器中配置，而应先制作镜像
  
  步骤
    编写Dockerfile，大小写
    FROM ubuntu
    RUN apt-get update
    RUN apt-get install -y nginx (自动确认安装
)
    EXPOSE 80
    
    在当前文件夹，执行docker build -t zyk/nginx .
      最后的点表示当前文件夹
    默认只能自己主机使用
      可以提交到DockerHub,或自己搭建DockerHub服务器
